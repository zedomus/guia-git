GitHub Pages

Publicar um site a partir do repositório (ex.: manual, documentação ou página do projeto).

🌐 Site ⚙️ Settings 🚀 Deploy

Passos

Objetivo: o site abrir automaticamente em ...github.io/... com o index.html.

1

Confirmar o ficheiro index.html

O Pages abre o index.html na raiz do repositório.

Confirma que tens: /index.html (na raiz do repo).
2

Ativar GitHub Pages

Settings → Pages → Deploy from branch.

Settings → Pages
Source: Deploy from a branch
Branch: main | Folder: /(root)
Depois: Save
3

Abrir o link do site

O GitHub mostra o URL do Pages.

O link será parecido com: https://TEU_USER.github.io/TEU_REPO/
Coloca esse link no README.
Prints sugeridos: (1) Settings → Pages configurado, (2) link do site.
