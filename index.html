Manual Interativo de GitHub para acompanhar uma PAP

Este site funciona como um manual de apoio para os alunos utilizarem ao longo do desenvolvimento dos seus projetos de PAP. A ideia é simples: o repositório GitHub passa a ser o “arquivo oficial” do projeto — com organização, evidências e histórico de trabalho.
